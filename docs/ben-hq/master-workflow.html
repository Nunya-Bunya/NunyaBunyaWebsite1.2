What Still Needs Manual Setup

These items can't be done via API and need you to click around in the admin UIs:

  1. Campaign Builder (mautic.nunyabunya.com → Campaigns → "NB Welcome Sequence")
  • Open the campaign, add the remaining 4 email send actions with delays (2d, 3d, 3d, 4d)
  • Publish the campaign
  1. Twenty CRM (crm.nunyabunya.com)
  • Create company: "Nunya Bunya" with domain nunyabunya.com
  • Create contact: Ben Conner (ben@nunyabunya.com)
  • Go to Settings → Developers → regenerate API key if needed
  1. n8n Workflows (n8n.nunyabunya.com)
  • Settings → API → create an API key
  • Create workflow: Mautic webhook → Create CRM contact
  • Create workflow: Mautic "score > 50" webhook → Slack #alerts
  • Create workflow: Schedule (Monday 9am) → Pull stats → Slack #wins
  1. Metabase Dashboard (reports.nunyabunya.com)
  • Open "Nunya Bunya — Marketing Overview" dashboard
  • Add cards: Mautic email stats, Matomo traffic, lead pipeline, Shlink clicks
  • Enable public sharing for client-facing link

Marketing Task Workflows

1. Onboard a New Client

StepWhereWhat to Do
1Sales PipelineCreate new Company + Contact. Move deal to "Won"
2Website AnalyticsAdd their website as a new Site (Settings → Websites → Add)
3Website AnalyticsCopy the tracking JS snippet → add to their website <head>
4Email & AutomationsCreate a new Segment for this client's leads
5Email & AutomationsCreate a landing page + form for lead capture
6Link TrackerCreate branded short links for their campaigns
7Social SchedulerAdd their social accounts
8ReportsCreate a client dashboard (connect to their data sources)
9BookingSet up a booking page for their consultations

2. Launch an Email Campaign

StepWhereWhat to Do
1Email & AutomationsEmails → New → choose template or build from scratch
2Email & AutomationsSet the segment (who receives it)
3Link TrackerCreate short links for any URLs in the email (for tracking)
4Email & AutomationsInsert the short links into the email body
5Email & AutomationsSend test email to yourself → verify links work
6Email & AutomationsSchedule or send immediately
7ReportsCheck campaign performance: open rate, click rate, conversions

3. Set Up a Drip Sequence (Welcome Series)

StepWhereWhat to Do
1Email & AutomationsCreate 3-5 emails (welcome, value, case study, offer, follow-up)
2Email & AutomationsCampaigns → New Campaign
3Email & AutomationsSet trigger: "Contact added to segment ☑"
4Email & AutomationsAdd decision points: "Email opened?" → Yes/No branches
5Email & AutomationsAdd delays between emails (e.g., 2 days, 3 days, 5 days)
6Email & AutomationsAdd action at end: "Modify lead score +10" if they engaged
7Automations HubCreate workflow: high-score lead → create deal in Sales Pipeline

4. Create a Landing Page + Lead Capture

StepWhereWhat to Do
1Email & AutomationsLanding Pages → New → use builder or paste HTML
2Email & AutomationsForms → New → add fields (name, email, phone, etc.)
3Email & AutomationsEmbed the form on the landing page
4Email & AutomationsSet form action: "Add to segment ☑" + "Add to campaign [Welcome Series]"
5Link TrackerCreate a short link to the landing page (e.g., go.nunyabunya.com/free-audit)
6Website AnalyticsSet up a Goal: "Visit thank-you page" = conversion
7ReportsTrack: form submissions, conversion rate, source attribution

5. Run a Social Media Campaign

StepWhereWhat to Do
1AI AgentsGenerate post copy + image suggestions for the campaign
2Social SchedulerCreate posts → attach images → set schedule
3Link TrackerCreate short links with UTM params for each platform
4Social SchedulerUse the short links in post copy
5Website AnalyticsMonitor traffic from social (Acquisition → Campaigns)
6ReportsDashboard showing: clicks per platform, conversions from social

6. Build a Client Report

StepWhereWhat to Do
1ReportsNew Question → select data source (Matomo/Mautic/CRM/Shlink)
2ReportsBuild visualizations: traffic chart, email stats, lead pipeline
3ReportsSave to a Dashboard → name it "[Client Name] — Monthly Report"
4ReportsSharing → Enable public link (for embedding) OR schedule email delivery
5ReportsSet up Subscription: auto-email the dashboard as PDF weekly/monthly

7. Track a Paid Ad Campaign (Google/Meta)

StepWhereWhat to Do
1Link TrackerCreate short links with UTM params: source=google&medium=cpc&campaign=spring-sale
2Use the short links as your ad destination URLs
3Website AnalyticsAcquisition → Campaigns → see traffic by UTM campaign
4Website AnalyticsSet up Goals for conversions (form submit, booking, purchase)
5Email & AutomationsLeads from ads auto-captured via forms → assigned to ad-specific segment
6ReportsDashboard: cost vs. leads vs. conversions per campaign

8. Score and Qualify Leads

StepWhereWhat to Do
1Email & AutomationsPoints → Define scoring rules:
+5 for email open, +10 for link click, +20 for form submit, +30 for page visit (pricing page)
2Email & AutomationsCreate segment: "Hot Leads" = score > 50
3Automations HubWorkflow: when lead enters "Hot Leads" segment → create deal in Sales Pipeline + Slack notification
4Sales PipelineWork the deal through the pipeline: Lead → Qualified → Proposal → Won

9. Automate a Workflow (n8n)

Common automations to set up:

AutomationTriggerAction
