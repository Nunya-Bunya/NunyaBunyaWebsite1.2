Review Queue Rules

  • NO auto-publishing — All content goes through team review
  • Ad copy — ALWAYS requires team approval. Never auto-deploy.
  • Negative reviews (1-3 star) — AI drafts response, team reviews before posting
  • Positive reviews (4-5 star) — Auto-responded in brand voice
  • Client reports — Team reviews before sending to client
  • Blog posts — Published as WordPress drafts, team sets live
  • Social posts — Queued in Buffer as drafts, team approves schedule

How to Add a New Client

  1. Open client_configs.json
  2. Copy the _template entry
  3. Fill in all fields (client_id, name, platforms, IDs, pipelines)
  4. Set "active": true
  5. Run onboarding: python pipeline_runner.py --onboard new-client-slug
  6. The pipeline runner will automatically include them in future scheduled runs

Automation Options

0 6 * * 1 cd /path/to/NunyaBunyaAgents/Execution && python pipeline_runner.py --day monday
