Growth OS — Agent Architecture

System Overview

Growth OS is a 3-layer AI agent hierarchy that covers the full marketing lifecycle:

Acquisition → Conversion → Nurture → Sales → Analytics → Expansion

Total agents: 50+ (43 existing + 7 new)

Infrastructure: Mautic, Twenty CRM, Matomo, Metabase, Shlink, Supabase, n8n

Layer 1 — Chief Operators (Strategy & Orchestration)

These agents make decisions. They read data from all sources and direct the execution layer.

AgentRoleInputsOutputs
Campaign Orchestrator (NEW)Plans multi-channel campaigns. Coordinates offline + online. Decides which agents to deploy.Client goals, budget, timeline, Metabase dataCampaign briefs, channel allocation, agent task list
Growth Strategist (existing: analyticsreviewagent)Reads all performance data. Identifies opportunities. Recommends strategic shifts.Matomo, Mautic, CRM, Shlink, ads dataStrategy recommendations, budget reallocation, channel priorities
Reporting Agent (existing)Synthesizes everything into client-facing reports.All data sourcesWeekly reports, AI insights

Self-Improving Loop

Performance Data (Matomo + Mautic + CRM + Shlink)
        │
        ▼
Growth Strategist Agent (reads all data)
        │
        ▼
Campaign Orchestrator (plans next actions)
        │
        ├──→ Ads Agent (adjust budgets, new copy)
        ├──→ Content Generator (new content based on what converts)
        ├──→ CRO Agent (improve pages that leak leads)
        ├──→ Creative Strategist (new hooks based on winners)
        └──→ Email Agent (adjust drip based on engagement)
                │
                ▼
        New Performance Data (loop restarts)
