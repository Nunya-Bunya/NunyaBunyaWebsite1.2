How This Works

  1. Each project gets a plan file in the current month's folder
  2. Plans use a simple task list with status markers: ☐ todo, ☑ done, ▶ in progress
  3. When you say "Let's work on X", Claude reads that project's plan and picks up at the current ▶ or next ☐ task
  4. At the end of a session, Claude updates the plan with progress
  5. At month's end, carry forward incomplete items to the next month's folder
