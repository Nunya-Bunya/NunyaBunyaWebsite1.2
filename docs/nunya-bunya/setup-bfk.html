Business Foundation Kit — Setup Guide

Last updated: 2026-04-15

Status: Phase 3 complete — delivery pipeline fully wired and tested

Architecture

Customer visits: nunyabunya.com/business-foundation-kit
        │
[50-question Survey] (7 steps, auto-saves to localStorage)
        │
Select Tier (Free lead magnet, $499 Plan + Style Guide, or $999 Foundation via Stripe)
        │
Submit Form → POST to n8n webhook: /business-foundation-kit
        │
        ├→ [Supabase] saves to bfk_submissions table
        │
        ├→ [Slack] posts "New BFK Survey" to #alerts
        │
        ├→ [Mautic] creates contact with tag: bfk-free
        │   └→ Enters "BFK Free Tier" segment → upsell campaign (4 emails over 10 days)
        │
        ├→ [If FREE]
        │   ├→ Returns {success: true} to browser → shows success screen
        │   └→ Triggers doc generation → HTTP POST to nb-task-server:8090/generate
        │       └→ bfk_generate_docs.py:
        │           ├→ Fetches submission from Supabase
        │           ├→ Calls Gemini 2.5 Flash to generate business plan HTML
        │           ├→ Wraps in branded A4 PDF template (WeasyPrint)
        │           ├→ Emails PDF to lead via AWS SES (with Book a Call CTA)
        │           ├→ Updates Supabase status → free_delivered
        │           └→ Posts "BFK Documents Delivered" to Slack
        │
        └→ [If PAID TIER — $499+]
            ├→ Creates Stripe Checkout Session
            ├→ Returns checkout URL → customer pays
            └→ Stripe webhook → n8n:
                ├→ Updates Supabase (payment_status: paid)
                ├→ Slack notification
                ├→ Creates contact in Twenty CRM
                └→ Triggers doc generation (21 documents)
