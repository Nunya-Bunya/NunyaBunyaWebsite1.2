This is the live operating queue. It is not a calendar, not an idea dump, and not a graveyard of finished work.

Rules

  • If it has an external due date, put it in DEADLINES.md.
  • If it is finished, move it to DONE.md.
  • If it is interesting but not active, move it to BACKBURNER.md.
