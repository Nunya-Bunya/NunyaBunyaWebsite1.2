Trend Intelligence Agent

Automatically find emerging content trends on social media and generate actionable viral content ideas.

A Python-based tool that scrapes trending content from TikTok, YouTube Shorts, Reddit, and Twitter/X to identify viral patterns and suggest content strategies for content creators and marketers.

🚀 Features

  • Multi-Platform Analysis: Scrapes trends from TikTok, YouTube, Reddit, and Twitter
  • Viral Pattern Detection: Identifies hooks, formats, and engagement patterns
  • Growth Velocity Tracking: Monitors trend momentum and early signals
  • Automated Reports: Daily trend reports with actionable content ideas
  • Modular Design: Separate modules for each social platform
  • Data Persistence: CSV storage with trend history
  • Notification System: Email and Slack integration for alerts

📋 Requirements

  • Python 3.8+
  • API credentials for Reddit (PRAW) and Twitter (Tweepy) - optional for demo mode
  • Internet connection for scraping

🛠️ Installation

  1. Clone the repository:
   git clone <repository-url>
   cd trend-intelligence-agent
  1. Install dependencies:
   pip install -r requirements.txt
  1. Configure API credentials (optional):
  • Create a config.json file with your API keys:
   {
     "reddit": {
       "client_id": "your_reddit_client_id",
       "client_secret": "your_reddit_client_secret"
     },
     "twitter": {
       "bearer_token": "your_twitter_bearer_token"
     }
   }

📖 Usage

Basic Trend Analysis

Run daily trend analysis for all platforms:

python src/main.py --run-daily

Platform-Specific Analysis

Analyze trends from a specific platform:

python src/main.py --run-daily --platform tiktok
python src/main.py --run-daily --platform youtube
python src/main.py --run-daily --platform reddit
python src/main.py --run-daily --platform twitter

Custom Output Location

Save trends to a custom file:

python src/main.py --run-daily --output my_trends.csv

Email Notifications

Send trend reports via email:

python src/main.py --run-daily --notify email --email your@email.com

Slack Notifications

Send trend reports to Slack:

python src/main.py --run-daily --notify slack --slack-webhook https://hooks.slack.com/...

View Help

python src/main.py --help

📊 Output Format

The agent generates CSV files with the following columns:

  • timestamp: When the trend was captured
  • platform: Source platform (tiktok, youtube, reddit, twitter)
  • name: Trend name or title
  • type: Trend type (hashtag, sound, video, meme, etc.)
  • virality_score: Predicted virality (high, medium, low)
  • sample_link: Link to sample content
  • suggested_idea: Actionable content idea
  • engagement_metrics: JSON string with engagement data

🔧 API Configuration

Reddit (PRAW)

  1. Go to Reddit Apps
  2. Create a new app (type: script)
  3. Copy clientid and clientsecret to config.json

Twitter (Tweepy)

  1. Go to Twitter Developer Portal
  2. Create a new app
  3. Get Bearer Token and add to config.json

⚠️ Important Notes

  • Rate Limits: Respect platform API rate limits to avoid being blocked
  • Terms of Service: Some scraping may violate platform terms - use official APIs when available
  • Demo Mode: The agent works without API keys using mock data for demonstration
  • Data Privacy: Only collects public trend data, no user personal information

🔄 Automation

Daily Cron Job

Add to crontab for daily execution:

# Run daily at 9 AM
0 9 * * * cd /path/to/trend-intelligence-agent && python src/main.py --run-daily --notify email --email your@email.com

Python Schedule

Use the built-in scheduler for testing:

import schedule
import time
from src.main import main

# Run daily
schedule.every().day.at("09:00").do(lambda: main(['--run-daily']))

while True:
    schedule.run_pending()
    time.sleep(60)
