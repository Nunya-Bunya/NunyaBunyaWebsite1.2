# Power Portraits — Style Guide

## Brand Identity

**Tagline:** Portraits that command attention.
**Positioning:** Premium headshot and portrait studio for professionals, creatives, and entrepreneurs in Brisbane.
**Personality:** Confident, refined, quietly luxurious. We're not a photo booth — we're a transformation experience.

---

## Signature Visual Element

**Full-bleed photography and editorial layout.** The product IS the image. Every Power Portraits touchpoint should feel like flipping through a high-end fashion or lifestyle magazine. Photography dominates. Typography is minimal and elegant. Whitespace (or rather, blackspace) lets the images breathe.

- Hero sections should be 60-80% photograph, with text overlaid minimally
- Gallery layouts: masonry grids, horizontal scroll strips, full-viewport images
- Before/after comparisons where appropriate (LinkedIn selfie vs Power Portraits headshot)
- Subtle parallax movement on images during scroll

**Never:** Clip art. Busy layouts. Text-heavy pages. Bright, flat colours. Anything that feels "template-y."

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Black (primary bg) | `#080808` | Page backgrounds |
| Deep | `#0e0e0e` | Section backgrounds |
| Card | `#141414` | Cards, form containers |
| Border | `#1e1e1e` | Subtle dividers |
| Gold (primary accent) | `#b8953f` | Accents, highlights, CTA text |
| Gold Light | `#C9A84C` | Buttons, hover states |
| White (warm) | `#f2ede8` | Headings, primary text (note: warm, not pure white) |
| Muted | `#6b6560` | Body text, secondary info (note: warm grey, not cool) |

**Rules:**
- Black backgrounds only. Power Portraits is a dark, premium brand.
- Gold is used sparingly — it's an accent, not a primary colour. Too much gold looks cheap.
- White text should be warm (`#f2ede8`), not cool/pure (`#ffffff`). This creates a film-like warmth.
- All borders and dividers should be barely visible — `#1e1e1e` against `#080808`.

## Typography

| Role | Font | Weight | Size | Style |
|------|------|--------|------|-------|
| Display / H1 | Cormorant Garamond | 300 (Light) | 36-52px | Mixed case, italic for emphasis |
| H2 | Cormorant Garamond | 400 (Regular) | 28-36px | Mixed case |
| Body | Montserrat | 300 (Light) | 14-15px | Sentence case, line-height 1.8 |
| Labels / Tags | Montserrat | 500 (Medium) | 10-11px | Uppercase, letter-spacing 0.2-0.3em |
| Navigation | Montserrat | 200-300 | 12-13px | Uppercase, letter-spacing 0.15em |

**Rules:**
- Headings are NEVER uppercase (unlike NB). Mixed case with italics for emphasis creates elegance.
- The `<em>` tag in headings should be italic AND gold-coloured. This is the PP signature.
- Body text is extremely light (300 weight). Dense text feels wrong for a luxury brand.
- Generous letter-spacing on all labels and navigation (0.15em+).
- Line-height for body text is 1.8 — more spacious than typical.

---

## Spacing & Layout

- **Container max-width:** 1000px (narrower than NB — more intimate)
- **Page padding:** 60px horizontal (desktop), 24px (mobile)
- **Section padding:** 80-100px vertical
- **Card padding:** 36-44px
- **Card border-radius:** 16px
- **Grid gap:** 80px (major layouts — generous, breathing room)

**Layout principles:**
- Photography is the primary layout element. Text wraps around and supports images.
- Two-column layouts: photography left, text/form right (or vice versa)
- Full-width image breaks between text sections
- Vertical rhythm is slow and spacious — scroll should feel contemplative, not rushed

---

## Photography Direction

- **Lighting:** Studio lighting with soft shadows, rim lighting, dramatic but flattering
- **Color grading:** Warm, slightly desaturated, filmic. Not punchy or oversaturated.
- **Composition:** Tight crops (head + shoulders), shallow depth of field, dark backgrounds
- **Mood:** Confident, approachable, powerful. Not stiff or corporate.
- **Editing style:** Subtle retouching — skin smoothing, eye enhancement, but still looks natural. No plastic/airbrushed look.

---

## Motion & Interaction

| Element | Behavior |
|---------|----------|
| Image reveals | Fade in with slight scale (1.02 → 1.0) on scroll |
| Gallery | Optional horizontal scroll with momentum |
| Hover on images | Subtle brightness increase (1.0 → 1.05) |
| Text reveals | Fade in from bottom, staggered by element |
| Transitions | Slow — 0.4-0.6s duration. Nothing snappy or bouncy. |
| Page feel | Smooth, contemplative, unhurried |

**No custom cursor.** That's NB's thing. PP uses the default cursor — the focus should be on the images, not the interface.

## Voice & Tone

**We sound like:** A luxury hotel concierge. Warm, confident, understated. We don't need to shout — the work speaks for itself.

**We say:** "Portraits that make you look as good as you are." / "Your image, elevated."
**We don't say:** "BOOK NOW! LIMITED SPOTS!" / "Best headshots in Brisbane!" / "Cheap headshots."

**Copy rules:**
- Short, elegant sentences. Never more than 2-3 lines per paragraph.
- Use the word "you" — it's personal, not institutional.
- Price is mentioned confidently, not apologetically. Premium pricing is a feature.
- Scarcity is communicated with restraint: "Limited spots each month" not "ONLY 3 LEFT!!!"
- Testimonials should feel like editorial quotes, not Amazon reviews.

## Component Patterns

**CTA Buttons:**
- Primary: gold text on transparent, with thin gold border. Understated.
- Hover: gold background fills in, text goes dark
- Shape: rounded (10-12px radius), generous padding

**Form Cards:**
- Dark card (`#141414`) with barely-visible border
- Heading: Cormorant Garamond with italic gold `<em>` element
- Labels: Montserrat uppercase, wide spacing
- Inputs: dark background, subtle border, warm white text

**Navigation:**
- Simple: logo left, location right. No complex nav needed for landing pages.
- Logo: "Power *Portraits*" in Cormorant Garamond, italic gold on "Portraits"
- Full nav (on main site): horizontal links, uppercase Montserrat, 200 weight

**Image Cards:**
- No border, no shadow — just the image with slight border-radius (8-12px)
- Caption below in muted text, Montserrat 300

## Do's and Don'ts

**Do:**
- Let photography dominate every page
- Use warm tones (warm white, warm grey, gold)
- Keep layouts spacious and breathing
- Use italic serif for emotional emphasis
- Make the experience feel luxurious, even on a $149 Portrait Day page

**Don't:**
- Use cool/blue-toned colours — PP is always warm
- Use uppercase headings (that's NB's aesthetic)
- Crowd the page with text — let images do the talking
- Use casual/slang copy — PP is refined, not hip
- Use the NB noise texture or custom cursor — those are NB-specific
