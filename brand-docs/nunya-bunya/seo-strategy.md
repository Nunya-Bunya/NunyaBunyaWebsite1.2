## Technical SEO Checklist

- [ ] Google Search Console verified for nunyabunya.com
- [ ] XML sitemap submitted
- [ ] robots.txt configured
- [ ] All pages have unique title tags and meta descriptions
- [ ] Schema markup on all pillar pages (LocalBusiness + FAQPage)
- [ ] Internal linking between pillar and cluster pages
- [ ] Page speed optimised (< 3 second load)
- [ ] Mobile responsive verified
- [ ] Image alt text on all images
- [ ] HTTPS everywhere

- [ ] Google Business Profile created and fully filled out

- [ ] 5+ Google Reviews (ask first clients)
- [ ] Location pages for Gold Coast, Ipswich, Toowoomba (future)

- **Week 1-2:** Publish all 3 pillar pages + 2 cluster blog posts
- **Week 3-4:** Publish 4 more cluster blog posts
- **Month 2:** Publish remaining cluster posts + start second round of content
- **Ongoing:** 2 new blog posts per month, targeting long-tail variations
